### Downloading SQLectron

1. Type on google sqlectron, then visit this page

<img src=https://user-images.githubusercontent.com/99172259/179120550-73e2d6cc-09ea-4b43-a952-5d3e399d1381.png width="400" height="200" />

2. Click **download GUI** and choose based on your desktop os. Here I chose:

<img src=https://user-images.githubusercontent.com/99172259/179120671-33a96012-b328-4ef9-9f8f-6862249490e2.png width="400" height="400" />

3. Extract the files and open SQLectron

### Connecting RDS to SQLectron

1. Configure the inbound rules used in the database security group

<img src=https://user-images.githubusercontent.com/99172259/179120910-b9b3ac9c-6f1a-4a1a-ad08-b9ebb3d34c81.png width="500" height="300" />

2. Open SQLectron and fill in the information needed. You can fill in the server address by copying the endpoint. User and password you need to fill in is the username and password you created when configuring RDS

<img src=https://user-images.githubusercontent.com/99172259/179121084-4afcd361-ce04-47fb-98fa-00fc68466e5f.png width="500" height="300" />

3. Test the connection, if it is successful, you are good to go.

<img src=https://user-images.githubusercontent.com/99172259/179121391-dfdbf533-9fdf-4e75-a45e-6e1b807fd68a.png width="400" height="200" />


4. Click save.
