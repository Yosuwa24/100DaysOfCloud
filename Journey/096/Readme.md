### Build AWS CDK Application

- Creating directories. Run this following command in the command prompt

       mkdir eks
     
       cd eks
     
       mkdir cluster cdk8s
     
       cd cluster

       cdk init app --language=python
       
 - Please make sure to run **python3 -m venv .venv** afterwards.
